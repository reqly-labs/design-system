# Reqly Design System

Design tokens, themes and brand assets for the Reqly ecosystem.

This repository contains the visual foundation shared across Reqly projects, ensuring a consistent experience between applications, websites and future tools.

## Includes

- Color tokens
- Typography definitions
- Spacing scales
- Border radius tokens
- Shadow tokens
- Theme configurations
- Logos and brand assets
- Mascot assets

## Usage

The design system is used across:

- Reqly
- Website
- Documentation
- Future Extensions and Tools

## Goals

- Consistent visual identity
- Reusable design foundations
- Centralized brand management
- Scalable design decisions
